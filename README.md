# 澪一 Bilibili沉浸式视频置顶

一个基于 WebView2 的 WPF 应用程序，为 Bilibili 视频观看提供沉浸式体验。

## ✨ 功能特性

- 🎬 **视频播放** - 链接 Bilibili 视频主站
- 🖱️ **鼠标穿透模式** - 隐藏 UI 元素，鼠标可穿透窗口
- 🔑 **全局快捷键** - 可自定义的全局热键
- 👔 **老板模式** - 隐藏窗口时自动暂停，显示时自动继续
- 💾 **配置持久化** - 自动保存用户设置和视频进度
- 🎯 **置顶窗口** - 始终保持在最上层

### 环境要求

- Windows 10/11 x64
- .NET 8.0 Runtime (已集成)
- WebView2 Runtime
